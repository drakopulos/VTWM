---KEYBINDS
super+1,2,3,4,5,6              navigate desktops
ctrl+alt+h,j,k,l               navigate desktops
ctrl+alt+up,down,left,right    navigate desktops
super+i                        show icon manager
super+u                        hide icon manager
super+h,j,k,l                  navigate icon manager
super+r                        restart VTWM
alt+tab                     cycle window ring
ctrl+alt+n                  pin/unpin window
super+up,down,left,right    snap windows 
super+F11                   fullzoom
super+c    clock
super+v    volume mixer
super+t    terminal
super+f    file manager
super+p    task manager
super+e    editor
super+w    web
super+m    media player

---Window Operations
button1 on root-window    open windowops menu
button3 on root-window    open main menu
button2 on root-window    open VTWM Windows
button1+alt on window|icon      move window or raise by holding alt and tapping button 
button3+alt on window|icon      move window or lower by holding alt and tapping button 
button3+super on window|icon    iconify/deiconify window
button1 on title    raise window
button3 on title    lower window
button1 on frame    move window
button3 on frame    resize window
button1 on icon    move or iconify
button3 on icon    windowops menu
Return on iconmgr           iconify
Return+super on iconmgr     iconify
button1+super on iconmgr    iconify
button1 on iconmgr          iconify
button3 on iconmgr          warp
button1 on virtual desktop    move screen or windows
